# bluetoothfiletransfer
# To implement a Bluetooth network with application as transfer of a file from one device to another.
Bluetooth is a short-range wireless communication technology that enables devices such as mobile phones, laptops, and peripherals to wirelessly communicate data or speech over short distances. Bluetooth's objective is to replace the cords that traditionally link devices while keeping their conversations safe.
